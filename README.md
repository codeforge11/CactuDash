# CactuDash
CactuDash is a minimalist and highly optimized dashboard designed for efficient server management and includes Docker.

## Supported Operating Systems

| OS              | Status |
|-----------------|:------:|
| Arch                |   ✅   |
| Debian/Ubuntu   |   ✅   |
| Raspbian        |   ✅   |
| Red Hat/Fedora  |   ✅   |
| Windows         |   ❌   |
